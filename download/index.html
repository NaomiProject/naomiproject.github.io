Install Naobian (Recommended)

  1. Download and install Etcher
  2. Download the Naobian image (.img file) for your system from the repo:
    3. Latest Naobian System Image

    Note: sha256 checksum if you want to check the download f308875fba9ffad8868aa256e50df095eaa2cf159adebd490c4a555ae147ee1c Naobian-1.0.zip

  3. Follow the Naobian Documentation

Manual Installation

  1. Follow the Config Documentation to setup the Audio Engine, Text-to-Speech, & Speech-to-Text.
  2. Fetch the repository
    3. curl -L "https://dl.bintray.com/naomiproject/rpi-repo2/stable/Naomi-2.2.zip" -o Naomi-2.2.zip
  3. Explode directory
    4. unzip Naomi-2.2.zip
  4. Rename directory
    5. mv Naomi-2.2 Naomi
  5. Go into the directory
    6. cd Naomi
  6. Setup the install
    7. chmod +x naomi-setup.sh
    chmod +x compile_translations.sh
  7. Run the install
    8. ./naomi-setup.sh
  8. Run the app
    9. python Naomi.py
